Directory Description

The data/GCJs folder contains processed feature files and ranking pair files for the GCJ-C I, GCJ-C II, GCJ-C++ I, GCJ-C++ II datasets, as described in Section V.A.1.
The data/obfuscations folder includes processed feature files and ranking pair files for the single and combined obfuscated datasets in Section V.D.1.
The data/aptmalware folder includes processed feature files and ranking pair files for the APT malware homology detection dataset in Section V.D.2 to V.D.4.
The data/Github folder includes processed feature files and ranking pair files for the Github dataset in Section I of our supporting documentation.

Extended experiments on realistic Github dataset

We introduce the Github data source to evaluate SepBIN's performance on realistic datasets more comprehensively. The experimental results are shown in Table I and Figure 1. On the Github dataset, the Precision@1 and Recall@5 metrics reach 85.13% and 75.16%, respectively, outperforming the Siamese-based baseline model by 8.06 and 6.48 points. And when utilizing different proportions of finetuning datasets, SepBIN maintains improvements compared to the baseline model. The gains in Precision@1 and Recall@5 metrics range from 2.14 to 11.13 points. Overall, the extended experiments on the realistic Github dataset further prove that SepBIN's network architecture and pretraining-finetuning mechanism can effectively transfer the feature separation ability, benefiting real-world binary authorship verification applications. The dataset collection process and experimental results are detailed in the document download link.
